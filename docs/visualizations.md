# Visualizations

Det har ar indexet for spelets visuella kartor.

Tanken ar att varje visualisering ska ligga dar den hor hemma, sa dokumenten inte blir for stora nar spelet vaxer.

## Visual Docs

- [Game and Town Flow](visual-game-flow.md)
- [Code Map](visual-code-map.md)
- [Story and Quest Progression](visual-story-progression.md)
- [Dungeon Maps](visual-dungeon-maps.md)
- [Combat and Leveling](visual-combat-and-leveling.md)

## When To Use What

- Use `Game and Town Flow` when debugging where the player should go next or how town navigation feels.
- Use `Code Map` when deciding which file to edit.
- Use `Story and Quest Progression` when adding quests, checking tier gates, or planning story pacing.
- Use `Dungeon Maps` when balancing dungeon pacing, rests, loot, keys, and boss routes.
- Use `Combat and Leveling` when changing action economy, crit fail, class features, or derived stats.
